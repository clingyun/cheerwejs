# CheerweJS

CheerweJS是一款基于bootstrap2.0风格的前端组件，力求让开发者使用最少的代码，完成预定义的功能需求，本框架没有什么高级的架构设计，没有太多新潮的设计理念，唯有最简洁的代码风格，让您快速上手使用


## 组件
    * Ajax
    * AutoComplete
    * Component
    * DatePicker
    * Dialog
    * Form
    * Grid
    * Mask
    * Message
    * NumberEditor
    * Pagebar
    * Select
    * Tabs
